Lookup Domains
==============

Does a whois and resolves a list of domains passed to it as arguments:

```
$ ./lookup_domains.py example.com example.org another.example.com
==============================================================================
example.com (A.B.C.D)
                    ns0.example.com (A.B.C.D)
                    ns1.example.com (A.B.C.D)

...

```
